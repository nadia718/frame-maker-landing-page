# Frame Maker Landing Page

A modern and responsive landing page developed by converting a Figma design into a fully functional Bootstrap website.

## Features

* Figma to Bootstrap conversion
* Fully responsive design
* Clean and modern user interface
* Mobile-friendly layout
* Reusable Bootstrap components
* Cross-browser compatibility

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

This project was created to practice converting a professional Figma design into a responsive Bootstrap landing page while maintaining design accuracy and responsiveness.
